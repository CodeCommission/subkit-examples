# Todos GraphQL - PouchDB example

[Live example](https://subkit-todos-api.cloud.dropstack.run/graphql)

## How to use locally

Install it and run:

```bash
npm install
npm run dev
```

## Deploy it to the cloud with [DropStack](https://dropstack.run)

```bash
npm i -g dropstack-cli
dropstack login
dropstack deploy
```
